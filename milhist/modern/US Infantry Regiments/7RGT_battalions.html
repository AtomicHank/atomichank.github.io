7th Infantry Regiment Battalions

Modern Era, United States, Infantry

This subpage gathers the battalion insignia associated with the modern 7th Infantry Regiment, the Cottonbalers. The local icon set includes the 1st, 2d, and 3d Battalions, while the public CMH battalion list currently documents the 2d and 3d Battalions.

1st Battalion

1st Battalion, 7th Infantry Regiment

The icon set preserves a 1st Battalion for the Cottonbalers, but the current public CMH infantry listing emphasizes the 2d and 3d Battalions rather than a public 1st Battalion page. This section is therefore best understood as a heritage placeholder in the icon set rather than as a fully reconstructed public battalion lineage entry.

2d Battalion

2d Battalion, 7th Infantry Regiment

The 2d Battalion carries the Cottonbalers' better-documented modern battalion lineage. It represents the regiment's long association with the 3rd Infantry Division, whose battalions kept the 7th Infantry's identity alive after the old regimental combat team era had passed and battalions became the Army's enduring maneuver units.

3d Battalion

3d Battalion, 7th Infantry Regiment

The 3d Battalion is another principal modern bearer of the regiment's history. Its significance lies in carrying the Cottonbalers' lineage forward inside the 3rd Infantry Division framework that had already defined the regiment in World War I, World War II, Korea, and later mechanized operations.

Research note: The 2d and 3d Battalion blurbs follow the regiment's public history and current CMH battalion listings checked on 24 March 2026. The 1st Battalion note reflects the difference between the local icon set and the public battalion list.
