5th Infantry Regiment Battalions

Modern Era, United States, Infantry

This subpage gathers the battalion insignia associated with the modern 5th Infantry Regiment, the Bobcats. The public CMH battalion list currently documents the 1st and 2d Battalions, while the local image set also includes a 3d Battalion icon.

1st Battalion

1st Battalion, 5th Infantry Regiment

The 1st Battalion is the regiment's best-known modern combat battalion. Its line ran through the Pentomic-era 1st Battle Group and then into the Vietnam-era 1st Battalion, 5th Infantry, which fought as a mechanized battalion with the 25th Infantry Division and gave the Bobcats one of their strongest modern battlefield identities.

2d Battalion

2d Battalion, 5th Infantry Regiment

The 2d Battalion carried the regiment's lineage into the late Cold War and post-Cold War force, reflecting the Bobcats' shift from a classic frontier and divisional regiment into a battalion-based lineage living inside modern brigades and divisions. Along with the 1st Battalion, it preserved the regiment's identity after the end of the old regimental combat team era.

3d Battalion

3d Battalion, 5th Infantry Regiment

The local artwork set includes a 3d Battalion icon, but the current public CMH infantry listing for the 5th Infantry Regiment documents only the 1st and 2d Battalions. This section therefore functions as a heritage placeholder inside the visual icon set rather than as a fully documented public lineage entry.

Research note: The 1st and 2d Battalion blurbs follow the regiment's public history and current CMH battalion listing checked on 24 March 2026. The 3d Battalion note reflects the difference between the icon set and the public battalion list.
