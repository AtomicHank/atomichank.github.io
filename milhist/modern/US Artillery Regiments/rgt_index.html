Overview

This index displays the current field artillery regiment symbol set in the same basic structure used for the infantry regiment section. Each regiment card opens a regiment hub page, and each regiment hub page links onward to a dedicated battalion page for that regiment's 1st through 5th Battalions.

The current set covers 56 field artillery regiments: the 1st through 42nd, then the 73rd, 75th through 84th, 86th, 92nd, and 94th Field Artillery Regiments.
