Overview

Unlike the Soviet airfields dataset, this workbook is a force-structure sheet with thousands of individual unit rows. These pages aggregate those rows into base locations so you can browse where formations were actually stationed and then drill into the command chain behind each site.

Use the map when you want geography first, and use the list when you already know the base name or host country you want to inspect.
